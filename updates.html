What are Patch Notes?

Patch Notes are updates that the devs add to the game usually they come very week, but the bigger ones that contain things such as agents, maps, and weapons take onger so they come out every 2 months. The weekly patch notes are usually balance changes to the game. For example, if an agent is deemed to strong for the community, the devs will make changes to the agent to make her still good, but balanced. The reason for this is because the devs want people to pick agents based on their play style and not because one is better than the other.
